How Latitude Relates to Humidity



This data set uses data reported at the same time across the globe. This means that the measurements were not taken at time relative to the location ( 09:00 GMT vs 09:00 local time) Humidity tends to be highest at night, when temperatures are lower. Because of this limitation, the true relationship latitude has with wind speed and humidity is unknown. Maximum or average scores could have a stronger relationship.
